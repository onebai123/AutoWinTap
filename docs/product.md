# AutoWinTap 产品文档

## 产品定位

**AutoWinTap** - AI 编程效率工具，你的窗口监管者

```
智能监控 | AI 分析 | 自动化操作
```

## 简介

AutoWinTap 是一款面向开发者的 AI 编程效率工具，通过实时监控多个 Windows 窗口（浏览器、编辑器、终端），结合 AI 分析能力，自动检测错误、提供调试建议、执行自动化操作。

## 核心价值

| 能力 | 说明 |
|------|------|
| 🔍 **自动化监控** | 实时监控多个开发窗口 |
| 🤖 **AI 分析调试** | 自动检测错误，AI 给出修复建议 |
| ⚡ **自动化输入** | 向窗口发送命令和按键序列 |

## 快速开始

### 1. 启动 Agent
```bash
cd agent
dotnet run
```

### 2. 启动 Server
```bash
cd server
npm install
npm run dev
```

### 3. 打开工作台
```
http://localhost:3000/workstation
```
